# Cheatsheet
## remote repos
cloning a repo/fork
~~~~bash
git clone git@github.com:USER-NAME/REPOSITORY-NAME.git
~~~~
pushing to the repo/fork
~~~~bash
git push origin main
~~~~

git add .
